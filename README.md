## Project Overview  
This project is a Digital Literacy Portfolio created as part of the CSE0001 course. The objective of this project is to develop essential digital skills such as creating professional online profiles, practicing safe internet usage, effective communication, and understanding cybercrime. The project consists of five tasks, each focusing on a different aspect of digital literacy, and all work is organized in a structured GitHub repository.

---

## Module Summaries  

### Task 1 – Digital Literacy Infographic  
Created an infographic using Canva covering key topics like digital literacy, safe internet practices, and email etiquette.

### Task 2 – Digital Portfolio  
Set up professional accounts on GitHub, LinkedIn, and Kaggle to build an online presence.

### Task 3 – Platforms Exploration  
Practiced coding on HackerRank and created a quiz using Google Forms for collaboration and data collection.

### Task 4 – Email Etiquette  
Drafted professional emails and created a checklist for responsible digital communication and social media usage.

### Task 5 – Cybercrime Awareness  
Developed a case study on cybercrime and created a prevention checklist to promote safe online practices.
